物理层

### 第一层: 物理层(Physical)
 
在物理介质上透明地传送原始比特流, `协议数据单元是bite` 

#### 第二层 数据链路层(Datelink)
+ 在物理线路上提供可靠的数据传输，使之对网络层呈现为一条无错，点到点的链路

+ 主要功能和服务
> + 物理地址
> + 成帧
> + 定界与同步
> + 差错控制 (差错的检测和恢复)
> + 流量控制
> + 信道的访问控制

#### 第三层 网络层(network)
+ 在源结点与目的节点间提供端到端的数据传输
+ 主要功能和服务
> + 路由选择
> + 数据分组转发
> + 流量控制与拥塞控制
> + 差错检测与恢复
> + 审计: 流量统计和记账
